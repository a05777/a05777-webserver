# AI-Powered High-Performance Web Server (Go-based)
# 一个AI写的高性能Web服务器（基于Go，人工调试修改）
系统要求：AMD64架构处理器，64位系统，支持systemctl，至少1个虚拟核心，1GB内存
测试环境：Debian12 and Ubuntu 22.04
HTTP端口填一个随机高位的即可，程序有协议嗅探，自动区分HTTP/HTTPS，只需要认真填写HTTPS端口

[![License](https://img.shields.io/badge/License-All%20Rights%20Reserved-red.svg)](LICENSE)

An ultra-efficient web server where the logic architecture was generated by AI, then fine-tuned and deployed by humans for real-world scenarios.
本项目逻辑架构由 AI 生成，并由人工进行场景调优与部署。

---

## 🌟 Project Highlights/info | 项目亮点/信息

* **Extreme Performance / 极致性能**: Achieved a measured throughput of **11,728 RPS** on a single-core (Xeon 6152) VPS environment.
  在 1C 核心（6152 处理器）的 VPS 环境下，实测吞吐量达到 **11,728 RPS**。
* **AI-Human Collaboration / AI 协作开发**: The core architecture was AI-generated, with manual optimization for specific edge cases and deployment.
  本项目逻辑架构由 AI 生成，并由人工进行场景调优与部署。
* **Minimalist Deployment / 极简部署**: Runs as a single binary without complex Nginx configurations; complete environment setup in under 2 minutes.
  单二进制文件运行，无需复杂的 Nginx 配置，2 分钟内完成全环境搭建。
* **Automatic Daemon / 自动守护进程**: Automatically generates `systemctl` daemon files for easy service management.
  自动生成 systemctl 守护进程文件。
